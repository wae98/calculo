    • VALOR ABSOLUTO

    El valor absoluto, denotado como |x|, es una función matemática que devuelve la distancia de un número real "x" al origen en la recta numérica, sin tener en cuenta su signo. En otras palabras, el valor absoluto de un número siempre es no negativo.

    Definición: Para un número real "x", el valor absoluto |x| se define de la siguiente manera:

    • Si x es positivo o cero, entonces |x| = x.
    • Si x es negativo, entonces |x| = -x, que es el opuesto de x.

    Ejemplos:

    1. El valor absoluto de 5 es |5| = 5, ya que 5 es positivo.
    2. El valor absoluto de -3 es |-3| = 3, ya que se toma el opuesto de -3.
    3. El valor absoluto de 0 es |0| = 0, ya que 0 está a una distancia de 0 del origen.

    El valor absoluto se utiliza comúnmente en matemáticas y ciencias para representar magnitudes, distancias y para simplificar expresiones algebraicas y resolver ecuaciones. También es útil en cálculo, geometría y en una variedad de aplicaciones en la vida cotidiana.
